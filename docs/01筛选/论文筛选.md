## 是否只保留 LLM OPD

结论：**基本是，只保留了 LLM / LLM-reasoning / LLM-adjacent OPD 论文**，但其中有 3 篇不是“纯文本 LLM-only”：

- `2604.03128` Self-Distilled RLVR：标注为 MLLM / RLVR，但研究的是 OPSD/RLVR 机制，和 LLM OPD 主线高度一致。
- `2605.03677` Uni-OPD：覆盖 LLM + MLLM，是统一 OPD recipe，仍属于 OPD 核心论文。
- `2603.11137` REOPOLD：标注包含 LLM / VLM / agentic，但主题是 relaxed OPD scaling reasoning，仍可作为 LLM OPD 算法论文阅读。

已剔除的 `X-OPD`、`VLA-OPD`、`Flow-OPD` 是更明显的 speech / robotics / text-to-image 扩展；`Nemotron-Cascade 2` 和 `MiMo-V2-Flash` 都是工业报告，但按你的剔除列表只移除了 NVIDIA，保留了 MiMo。

## 最终检查

剔除后剩余论文均可归入以下集合之一：

- **LLM OPD / OPSD / RLVR distillation**
- **LLM reasoning distillation**
- **LLM self-distillation**
- **LLM-adjacent OPD**, 如 MLLM 或 agentic reasoning，但仍围绕 OPD/RLVR 方法本身

因此，过滤后清单已经基本满足“只保留 LLM OPD”的目标。如果要进一步收窄为 **纯文本 LLM-only**，建议额外剔除 `2604.03128`、`2605.03677`、`2603.11137`。
