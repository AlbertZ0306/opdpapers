# OPD 相关论文类型总结

更新时间：2026-06-19  
对象：`论文索引.md` 中整理的早期 31 篇 OPD / OPSD / RLVR distillation 相关论文，以及 `2026年5月以来OPD增量总结.md` 中新增的 84 篇 LLM/agentic/coding OPD 方法与分析、4 篇模型/系统技术报告。

## 类型定义

- **新算法 / 新训练框架**：核心贡献是提出新的 OPD、OPSD、RLVR distillation 或相关训练算法。
- **机制性 / 分析性实验**：核心贡献是通过现象、ablation、指标或理论解释失败模式/成功条件；可能也给出 recipe，但分析是主线。
- **工程系统 / 技术报告**：核心贡献是把 OPD 放进大规模 post-training pipeline 或产品级模型训练流程。
- **综述 / taxonomy**：整理已有方法和谱系，不以新算法为主。
- **扩展应用**：把 OPD 思路迁移到多模态、VLA、flow matching、隐私等非标准 LLM OPD 场景。

## 更新后的类型判断

2026-05+ 增量之后，OPD 论文已经不只是“新算法堆叠”，而是明显分化成几条研究线：

1. **机制分析正在前移**：`Unmasking OPD`、`Many Faces`、`Post-Training is About States`、`Geometry of OPD` 等论文说明，后续更可能围绕 state distribution、teacher signal reliability、token teachability 来写。
2. **长轨迹可靠性成为主 failure mode**：Prune-OPD、SFD、Early Stopping、ADWIN、KL Agreement Trap 都在处理 prefix drift / teacher supervision decay。
3. **选择性监督成为算法主线**：TRACE、Token Teachability、Position-Weighted OPSD、Path-Conditioned SD、Filter-Then-Reweight 都在问“哪些 token/step 该更新”。
4. **OPD 与 RLVR 的分工更清晰**：Sparse-to-Dense Reward Principle、Self-Supervised OPD、CAST、RLCSD、Self-Conditioned Credit Assignment 都把 RL 的探索和 OPD 的 dense credit 组合起来。
5. **OPD 信号从 logits 扩展到语义反馈**：OmniOPD、Rubric-conditioned SD、Constitutional OPD、language feedback、evidence-guided OPD 说明 black-box / semantic OPD 会继续扩大。
6. **Agentic OPD 成为独立分支**：医疗、多轮对话、搜索、KBQA、multi-turn tools 中，OPD 的核心对象从 token-level reasoning trace 扩展到 turn / tool / trajectory-level supervision。
