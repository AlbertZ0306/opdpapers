## 0. 概览

### 0.1 本研究要测量的两个核心量

整套实验围绕两条相互独立的证据链展开。

**第一，复现率 $\rho$**，定义为：

$$
\rho=\frac{\mathrm{Acc}(\text{无教师方差缩减})-\mathrm{Acc}(\text{GRPO})}{\mathrm{Acc}(\text{OPD})-\mathrm{Acc}(\text{GRPO})}.
$$

它衡量教师带来的提升中，有多大比例能被一个"零外部知识"的方差缩减方法复现。

**第二，覆盖集解锁量**，即 $S_{\text{OPD}}\setminus\big(S_{\text{GRPO}}\cup S_{\text{无教师VR}}\big)$ 的大小——OPD 是否解出了 GRPO 与无教师方差缩减都解不出的题目（在较大采样次数 $k$ 下衡量）。

**判读规则**：若 $\rho$ 接近 1 且覆盖集不扩大，支持"方差缩减假说"（结论偏向去魅，称结论形态 A）；若 $\rho$ 接近 0 且覆盖集扩大，支持"知识迁移假说"（结论偏向正面，称结论形态 B）。

### 0.2 实验的先后顺序（每阶段附判定门）

各阶段记为 Phase，判定门记为 G。

```text
Phase −1  预备与口径校准          ── 判定门 G−1：训练框架/验证器/评测口径/噪声地板全部达标
   │
Phase 0   基础设施 + 第零层零训练协方差判断 ── 判定门 G0：教师信号是否像一个方差缩减控制变量
   │
Phase 1   建立 ρ 的分母：GRPO + OPD（自蒸馏，最小规模） ── 判定门 G1：Δ=Acc(OPD)−Acc(GRPO) ≥ 3σ
   │
Phase 2   测量主结果（E1）：加入无教师方差缩减，算 ρ 与覆盖集 ── 判定门 G2：ρ 稳定且方差确降
   │
Phase 3   相图实验（E2）：沿教师强度 / 轨迹正确性 / 分布内外三轴
   │
Phase 4   机制确认（E3）：方差、梯度信噪比、不动点漂移
   │
Phase 5   对照定位（E4）：把 RLSD、vOPD 投到 ρ 谱系；并在代码域验证泛化
```

**Phase −1 是主实验前的必经预备**（详见第 4.0 节）：它不产出论文数字，只确保后续所有准确率、$\Delta$、$\rho$ 的测量都是可信的——即训练框架正确、验证器干净、评测口径固定、噪声水平已知、算力对齐口径已定。Phase 0 至 Phase 2 同时构成课题正文所述的三层可行性预验证；跑通即可得到核心数字。Phase 3 至 Phase 5 把单点结论扩展为可发表的相图与机制证据。

## 1. 实验对象（方法与条件集）

所有条件**共享同一基座模型、同一题目集合、同一算力预算**，仅改变信用信号的来源。这是 $\rho$ 具有合法含义的前提。

| 代号 | 条件 | 信用信号来源 | 是否注入外部知识 | 角色 |
|---|---|---|---|---|
| **C-GRPO** | GRPO | 轨迹级可验证奖励 + 组内基线 | 否 | $\rho$ 的分母基准 |
| **C-OPD-self** | 自蒸馏 OPD | 教师对数比（教师 = 冻结初始权重或参数滑动平均自教师） | 否（同一模型） | 核心靶标：主流 OPD |
| **C-OPD-s2w** | 强到弱 OPD | 教师对数比（教师确实更强） | 是 | 实验 E2 中 $\delta<0$ 的一臂 |
| **B1** | 学习型价值头 | $R_i-V_\phi(s_{i,t})$，稠密逐词元 | 否 | 无教师方差缩减 |
| **B2** | 无教师控制变量 | $A^{\mathrm{RL}}-\beta^{*}(\hat m-\bar m)$，$\hat m$ 仅用学生自身特征 | 否 | 无教师方差缩减（最干净） |
| **B3** | 自洽性稠密信号 | 学生自身一致性 / 熵正则的稠密信号 | 否（定义上不引入新知识） | 无教师方差缩减（上界对照） |
| **B4** | CAST（`2606.00172`） | 不依赖答案、停止梯度的自教师塑造词元级优势 | 否 | 收编的现成无教师基线 |
| **R-RLSD** | RLSD（`2604.03128`） | 方向取自 RLVR、力度取自教师 | 是 | 实验 E4 定位用 |
| **R-vOPD** | vOPD（`2605.07865`） | 教师反向 KL 控制变量 | 是 | 实验 E4 定位用 |

无教师方差缩减基线（B 系列）须遵循的原则：**只用学生模型自身的价值、统计量或分布来降低方差，不注入任何外部知识。** 唯有如此，"这些基线能复现 OPD 的提升"才能合法地推出"OPD 的提升并非来自知识"。$\rho$ 取 B1 至 B4 中表现最好的一个（只要任一基线复现，即支持方差缩减假说）；同时报告全部，互为消融。基线 B2 有一个理论锚点：vOPD 已证明 OPD 的最优控制变量是教师的逐词元反向 KL，因此 B2 的无教师预测器越接近这一量越好。

## 3. 评测指标定义

| 指标 | 定义 | 用途 |
|---|---|---|
| 准确率 Acc（avg@$k$） | 固定 $k$（如 avg@16）下的平均正确率 | $\rho$ 的三个端点 |
| **复现率 $\rho$** | $\dfrac{\mathrm{Acc}(B^{*})-\mathrm{Acc}(\text{GRPO})}{\mathrm{Acc}(\text{OPD})-\mathrm{Acc}(\text{GRPO})}$，$B^{*}$ 为表现最好的无教师基线 | 实验 E1 的核心数字 |
| 可解题目集合 $S_{\text{方法}}$ | 在 pass@$k_{\max}$ 下可解出的题目集合 | 实验 E1 的第二证据 |
| 解锁量 | $\lvert S_{\text{OPD}}\setminus(S_{\text{GRPO}}\cup S_{\text{无教师VR}})\rvert$ | 实验 E1 / E2 |
| 信用方差 | 逐词元信用估计的方差 | 实验 E3 机制 |
| 梯度信噪比 | $\lVert\mathbb{E}[g]\rVert / \mathrm{std}(g)$ | 实验 E3 机制 |
| 与教师的 KL 散度 | $\mathrm{KL}(\pi_\theta\Vert\pi_T)$ 随训练的演化 | 实验 E3：检验不动点是否被拉向教师 |
| 教师偏差估计 $\hat\delta$ | 理论附录给出的教师偏差估计 | 实验 E2：与 $\rho$ 的负相关 |

## 4. 主实验逐项设计

### 4.0 Phase −1：预备与口径校准（主实验前必做）

**目的**：主实验的所有结论都建立在"准确率、$\Delta$、$\rho$ 测得准"这一前提上。本阶段不产出论文数字，只逐项排除会**悄然污染**整条实验链的隐患。任何一项不达标，后续的 $\rho$ 都不可信。每项都是一个带判定标准的小实验，而非单纯的工程搭建。

| 代号 | 预备实验 | 为何必须前置 | 通过标准（G−1 的子门） |
|---|---|---|---|
| **P−1 训练框架复现性** | 在一个**已知设置**上复现一篇公开论文的 GRPO 与自蒸馏 OPD 结果（如 `2602.12125` / OPSD `2605.06188` 的某个数据点） | 若 GRPO 实现偏弱或有缺陷，分母 $\Delta$ 与 $\rho$ 全部失真；必须先证明框架能产出文献量级的结果 | 复现的 avg@$k$ 落在公开值的合理容差内（如 ±2–3 个百分点） |
| **P−2 验证器审计** | 在带标注的题目子集上测量奖励验证器的假阳性率、假阴性率，修正等价形式（如 $\tfrac{1}{2}$ 与 0.5、单位、LaTeX 变体） | 验证器的错误会同时污染准确率和优势信号 $A^{\mathrm{RL}}$，直接毒化第零层协方差与全部 $\rho$ | 假阳性率、假阴性率均低于阈值（如 < 1–2%）；等价形式匹配修复完成 |
| **P−3 评测口径校准** | 固定解码参数（温度、top-p、最大长度）；验证 pass@$k$ 随 $k$ 单调、avg@$k$ 跨种子稳定；**通过试验选定覆盖集的 $k_{\max}$**（pass@$k$ 的饱和点） | 覆盖集探针强烈依赖 $k_{\max}$；$k$ 太小测不出解锁、太大浪费算力。评测口径不固定则跨方法不可比 | pass@$k$ 在 $k_{\max}$ 之前饱和；avg@$k$ 的种子方差可接受；口径写定为协议 |
| **P−4 教师有效性** | 检查自蒸馏教师的对数比分布非退化（教师对数比 $c^{\mathrm{OPD}}$ 方差大于零，否则 OPD 近似无操作）；对强到弱蒸馏，确认教师在评测集上**确实更强**且学生有提升空间 | 若自蒸馏教师信号近乎为零，OPD 与 GRPO 无差异，分母塌缩；若强教师并不更强，$\delta<0$ 一臂不成立 | 自蒸馏教师对数比非平凡；强到弱蒸馏中 Acc(教师) 显著高于 Acc(学生) |
| **P−5 算力对齐口径** | 定义并试验"对齐算力预算"的**单位**：按梯度更新步数 / 按学生采样词元数 / 按实际耗时；显式处理 OPD 多出的教师前向开销 | 第 5 节要求三条件算力对齐，否则 $\rho$ 被"方差缩减训得更久、算得更多"混淆；OPD 与 GRPO 的算力构成不同，必须先定口径 | 形成一份可执行、且经试验验证的对齐约定 |
| **P−6 噪声地板与统计功效** | 用 GRPO 跑至少 3 个随机种子，测量准确率的种子标准差 $\sigma$；据此估计可检测的最小 $\Delta$，反推判定门 G1 所需的种子数 | 不知道 $\sigma$ 就无法判断 G1 的"$\Delta\ge3\sigma$"是否可达；有可能整个研究在此规模上根本测不出现象 | 预期的 $\Delta$ 在计划的种子数下可被检测；否则增加种子或提升规模 |

**复用约定**：P−1 与 P−6 都需要训练 GRPO，这条 GRPO 训练**直接复用为 Phase 0 协方差判断的输入**，不重复训练。P−3 选定的 $k_{\max}$、P−5 定下的算力单位、P−2 修好的验证器，在后续所有阶段沿用。

**判定门 G−1**：六个子门全部通过，方可进入 Phase 0；任一不通过，先修复该项（修验证器、调评测、换教师、重定算力口径、加种子、修框架），**不要带着有问题的口径进入主实验**。

### 4.1 Phase 0：基础设施 + 第零层零训练协方差判断

**目的**：搭建三条件共用的 GRPO/OPD 训练器；在投入大量算力之前，用纯前向推理预判 $\rho$ 的方向。

**步骤**：
1. 实现或复用一个 GRPO 训练器（DAPO-Math-17K，Qwen3-1.7B-Base，组采样）。训练**一条** GRPO 至收敛。
2. 取其采样轨迹，对选定教师做**纯前向推理**，逐词元计算教师对数比 $c^{\mathrm{OPD}}=\log\pi_T-\log\pi_\theta$ 与强化学习优势 $A^{\mathrm{RL}}$。
3. 测量协方差 $\mathrm{Cov}(A^{\mathrm{RL}},c^{\mathrm{OPD}})$、方差 $\mathrm{Var}(A^{\mathrm{RL}})$，以及把 $A^{\mathrm{RL}}$ 对 $c^{\mathrm{OPD}}$ 回归所得的判定系数 $R^2$（即教师作为控制变量所能吸收的方差比例）。

**判定门 G0**：
- $R^2$ 或协方差**高** → 教师信号在形态上即为方差缩减控制变量 → 预示 $\rho$ 偏高，进入 Phase 1 时信心更足；
- 协方差**接近零** → 教师信息与强化学习信用正交 → 预示 $\rho$ 偏低，仍可进入但预期走向结论形态 B；
- 取不到稳定的优势信号 → 先修复训练器或方差测量口径。

**成本**：1 条 GRPO 训练 + 若干前向推理，无需训练 OPD 或方差缩减基线。

### 4.2 Phase 1：建立 $\rho$ 的分母（确认现象是否存在）

**目的**：在最小规模上确认"OPD 相对 GRPO 存在稳定、可测、超过种子噪声的提升"——没有它就没有可归因的对象。

**步骤**：在 Qwen3-1.7B-Base、DAPO-Math-17K 上，于**严格相同的采样数、更新步数、总词元预算**下，分别训练 C-GRPO 与 C-OPD-self，每个至少 3 个种子（核心数字建议 5 个）。在 MATH500/AMC23/AIME24/25 上测 avg@16，计算 $\Delta=\mathrm{Acc(OPD)}-\mathrm{Acc(GRPO)}$ 及其种子标准差 $\sigma$。

**判定门 G1**：
- $\Delta\gtrsim 3\sigma$ → 分母稳健，进入 Phase 2；
- $\Delta$ 淹没在噪声中 → 小幅提升规模（改用 Qwen3-4B，或换一个更易出差距的条件），或承认"此规模无可归因提升"并**停止**（切勿在分母近零时计算 $\rho$）；
- 条件选择：优先自蒸馏（核心靶标）；若其 $\Delta$ 太小，可先用强到弱蒸馏拿一个干净的分母把流程跑通（代价是该条件下 $\rho$ 偏低本在意料之中）。

---

### 4.3 Phase 2：主结果（实验 E1）——复现率与覆盖集

**目的**：得到本研究的核心数字。

**步骤**：
1. 复用 Phase 1 的训练器，加入无教师方差缩减基线 B1/B2/B3/B4（同模型、同题目、同预算、至少 3 个种子）。可行性预验证阶段可只先上最便宜的 B1 或 B4（CAST）；正式实验则全部报告。
2. 用表现最好的基线 $B^{*}$ 计算 $\rho$；
3. 测量 pass@$k$（$k\in\{1,4,16,64,256\}$）得到各方法的可解题目集合 $S$，计算解锁量 $\lvert S_{\text{OPD}}\setminus(S_{\text{GRPO}}\cup S_{\text{无教师VR}})\rvert$；
4. **同时**用 Phase 0 的口径确认这些基线确实把信用方差降到了 OPD 的量级。

**判定门 G2**：
- $\rho$ 稳定（种子间方差小）且基线方差确实降到 OPD 量级 → 结果可信，进入 Phase 3；
- $\rho$ 偏高但基线方差未下降 → **假阳性**：基线只是恰好刷高了分，返回修基线，不要下结论；
- $\rho$ 噪声大 → 返回 G1 检查分母，或增加种子。

**预期产物**：一句核心结论——"在主流自蒸馏数学 OPD 上，无教师方差缩减复现了 $\rho$ 比例的提升，覆盖集（不）扩大"。

### 4.4 Phase 3：相图实验（实验 E2）

**目的**：把单点的 $\rho$ 扩展为"$\rho$ 在何种条件下高、何种条件下低"的可证伪相图（对应课题正文的预测）。沿三条轴各测一组 $\rho$ 与解锁量：

| 轴 | 低 $\rho$ 一端（知识迁移） | 高 $\rho$ 一端（方差缩减） | 实现方式 |
|---|---|---|---|
| 教师强度 $\delta$ | 强到弱蒸馏（C-OPD-s2w，$\delta<0$） | 自蒸馏（C-OPD-self，$\delta\approx0$） | 更换教师：Qwen3-8B/30B-A3B 对比冻结自教师 |
| 轨迹正确性 | 错误轨迹占多数 | 正确轨迹占多数 | 按组内正确率分桶训练/评测 |
| 分布内外 | 分布外（OlympiadBench/GPQA/MMLU-Pro） | 分布内（MATH500/AMC23） | 更换评测集 |

**预测与检验**：$\rho$ 在第一类条件下低、第二类条件下高；且与理论判据中的教师偏差估计 $\hat\delta$ 的量级**负相关**（同时测量 $\hat\delta$）。

**判定门 G3**：相图若与预测一致 → 得到强结构化结论；若紊乱无规律 → 说明 $\rho$ 受未受控的混淆因素主导，返回检查 Phase 2 的控制变量（预算与方差匹配）。

---

### 4.5 Phase 4：机制确认（实验 E3）

**目的**：把"$\rho$ 的高低"与"方差是否真降、不动点是否真移动"对应起来，排除"$\rho$ 由其他因素驱动"的可能。

**步骤**（在 Phase 2/3 已有的训练上追加测量，不新增训练）：
1. **方差侧**：测量各方法的信用方差与梯度信噪比；验证无教师方差缩减是否把方差降到与 OPD 相同的量级（这是方差缩减假说成立的必要条件）。
2. **不动点侧**：测量与教师的 KL 散度 $\mathrm{KL}(\pi_\theta\Vert\pi_T)$ 随训练的演化——OPD 是否真把不动点拉向教师，而 GRPO 与方差缩减基线则不会。
3. **对账**：$\rho$ 高 应当对应（方差降足 且 不动点未显著移动）；$\rho$ 低 应当对应（不动点被拉向教师 且 出现解锁题目）。

**判定门 G4**：机制与 $\rho$ 自洽 → 因果叙事成立；不自洽（例如 $\rho$ 高但不动点大幅移动）→ 重新审视"方差缩减对比知识"的可操作定义。

---

### 4.6 Phase 5：对照定位与泛化（实验 E4）

**目的**：把已有的"力度类"方法定位到 $\rho$ 谱系上，并验证结论可跨领域成立。

**步骤**：
1. 在同一套设置下训练 R-RLSD、R-vOPD，测量它们的 $\rho$，说明这类方法本质落在方差缩减区（预期 $\rho$ 偏高）。
2. **第二领域泛化**：在代码领域（Eurus-RL-Code / DAPO-code 训练，LiveCodeBench v6 / MBPP+ / HumanEval+ 评测）重跑实验 E1 的核心（GRPO、自蒸馏 OPD、最强方差缩减基线），观察 $\rho$ 与覆盖集结论是否保持。

**判定门 G5**：跨领域一致 → 结论稳健，可写"在所测条件下普遍成立"；出现分歧 → 明确限定适用领域，把"在何处成立"本身作为贡献。

---

## 5. 全局控制与公平性（贯穿所有阶段）

1. **算力对齐**：所有条件固定相同的题目数、每题采样数 $G$、更新步数、总训练词元数。否则 $\rho$ 会被"方差缩减训得更久"混淆。
2. **随机种子**：每个条件至少 3 个种子（核心的 $\rho$ 与相图关键点用 5 个）；一切差值报告均值与标准差。
3. **$\rho$ 作为"差值之比"的风险**：仅在 $\Delta\ge3\sigma$（即通过 G1）时才采信 $\rho$；并报告 $\rho$ 的自助法（bootstrap）置信区间。
4. **评测协议统一**：相同的解码温度、相同的 $k$、相同的验证器；覆盖集使用同一固定题目子集。
5. **"零外部知识"的边界**：基线 B1、B2 完全不含任何教师、更强模型或更优分布；基线 B3、B4 仅使用学生自身分布，单独标注其"上界/收编"性质。
6. **时效性**：投稿前重新运行一次新颖性核查（领域进展极快）。

风险与应对的完整列表见课题正文第 10 节（基线不公平、零知识边界、$\rho$ 幅度巧合、规模外推、ProRL 争议前提、与 CAST 构造重叠）。

## 6. 实验依赖关系与最小可发表切片

```text
Phase −1 (G−1) ──► Phase 0 (G0) ──► Phase 1 (G1) ──► Phase 2 (G2) ──► Phase 3 (G3)
                                                          │                  │
                                                          └──► Phase 4 (G4) ◄┘
                                                                             │
                                                                      Phase 5 (G5)
```

- **Phase −1 是全链前置**：训练框架、验证器、评测口径、噪声地板未达标前，不进入 Phase 0；它不进论文，但决定后续一切测量是否可信。
- **最小可发表切片**：Phase −1（预备）+ Phase 0 至 Phase 2（实验 E1）+ Phase 4（实验 E3 机制确认）。即"复现率 + 覆盖集 + 机制确认"三件套，已可构成一篇去魅型经验论文。
- **完整顶会版本**：再加 Phase 3（相图）+ Phase 5（定位与跨领域），把单点结论升级为"OPD 收益归因的条件地图"。
- **任一判定门触发"停止/转向"**：G1 失败意味着现象不存在，转向候选课题 P1（因果信用审计）；G2 或 G4 判定 $\rho$ 低且覆盖集扩大，则切换到课题的结论形态 B（正面叙事，同样可发表）。
