# opdpapers — OPD 论文库与研究规划仓库

这是一个**研究文档仓库**（非代码库），用于整理 OPD / OPSD / RLVR+distillation 相关论文、本地 PDF、失败模式图谱，以及由此衍生的研究想法与实验规划。文档与文件**优先中文命名**。完整导航见 `README.md`。

## 目录速览

| 路径 | 内容 |
|---|---|
| `docs/00索引/` | 全量论文索引（类别、来源链接、本地 PDF 路径） |
| `docs/01筛选/` | 论文筛选总结、按机制/类型分类 |
| `docs/02图谱/` | **事实层**：逐篇 failure mode 映射、研究现状、baseline/loss 对照、模型数据评测统计 |
| `docs/03问答/` | 论文阅读问答记录（逐篇解释、loss/baseline 讨论） |
| `docs/04想法/` | **想法层**：研究方向、候选课题、实验设计、执行规划（见下） |
| `docs/90元数据/` | papers.bib、下载记录、增量候选 JSON |
| `papers/` | 论文 PDF（`llm_opd_methods/`、`model_tech_reports/`、`multimodal_and_adjacent/`、`local_originals/`） |
| `official_pages/` | 官方模型发布页的本地保存 |

## 核心约定

1. **事实与想法分离**：`02图谱/`、`03问答/` 放论文事实与逐篇总结；`04想法/` 放研究想法与实验设计，**不要把逐篇事实总结混进 04想法**，也不要把主观推测混进图谱。
2. **本地优先回答**：本地已有 PDF 与文档时，优先基于本地论文和现有文档回答，除非确实需要最新外部信息。需要外部信息时，新颖性核查走 deep-research（带引用、对抗式核验）。
3. **新增论文的更新流**：先更新 `docs/00索引/论文索引.md` 与 `docs/01筛选/论文筛选.md`，再更新相关图谱文档；新增文件后在 `README.md` 索引表登记一行。
4. **arXiv 编号**：正文中用反引号包裹（如 `2605.06188`）。注意库中不少编号是 2026 年（2604/2605/2606）日期，属较新或内部占位，引用时以核查结论为准。
5. **中文命名**：新文档用中文文件名，与现有风格一致。

## 04想法 / 当前活跃研究线（P5）

当前主线是候选课题 **P5：OPD 的收益是知识迁移还是方差缩减**。相关文档构成一个体系：

- `研究方向与候选想法.md` —— 总纲：候选总表、P 系列（P1–P5）、演化记录。
- `OPD收益归因_方差缩减还是知识迁移.md`（速记版）/ `_正式版.md`（书面、含术语定义）/ `_逐段解释.md` —— P5 课题正文。
- `OPD收益归因_实验设计.md`（速记版）/ `_正式版.md` —— 主实验设计（E1–E4 + 前置闸门）。
- `OPD收益归因_执行路线图.md` —— 从第 1 步到投稿的 16 步时间有序规划。
- `OPD方差偏差分解理论.md` —— 理论附录（control-variate 偏差-方差分解）。
- `方向力度解耦想法.md` —— 实验臂与诊断指标。

> **正式版/速记版成对存在**：内容一致，正式版易读、速记版紧凑。改动其一时，确认是否需要同步另一版（默认问用户改哪套、是否同步）。
